Información Importante
ID Público

Cada cliente recibe un ID público único (UUID) que se usa para identificarlo de manera segura en la API y el widget web.

Almacenamiento

Todos los documentos y vectores se almacenan en PostgreSQL de manera segura y con aislamiento completo entre clientes.

Multiidioma

El sistema detecta automáticamente el idioma de las consultas y responde en el mismo idioma (Español, Inglés, Francés, etc.).

Seguridad

Cada cliente tiene acceso solo a sus propios documentos. No hay cruce de información entre diferentes empresas.
